<h1>Music App</h1>
<p>Single Page Application created with Vanilla JS</p>

<p>Usage:</p>

<p>To start the server</p>
<p>cd server</p>
<p>node server.js</p>

<p>To start the frontend</p>
<p>npm start</p>


<p>To start the tests:</p>
<p>cd tests</p>
<p>npm run test</p>
